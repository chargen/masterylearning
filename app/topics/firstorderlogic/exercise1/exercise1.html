PL1 Terms and Signatures

In this exercise you learn to understand Terms and Signatures of first order logic. Sometimes also called PL1.

Given a set `VAR={x, y}` and `Sigma = {vv, ^^, not}`, please decide, which of the following statements are true

  • `x` is an atom.
  • `x` and `not x` are both literals.
  • "`x ^^ y`" is a term.
  • Non of the above.

Storyboard

Goal: In this exercise the student learns to understand Terms and Signatures of first order logic. Sometimes also called PL1.

Automation: This exercise can be automated in the following way:

  • We could randomly initialize a set of VAR and `Sigma` and then combine elements of the set to form terms, atoms, literals and also wrong expressions.
  • For each generated expression we store if its correct or wrong and then insert the expression into a template string that matches their type.
  • Templates would then be: "%s is an atom", "%s and %s are both literals.", "%s is a term." plus the generic "Non of the above." that is always present.

Usage: Please select option 3.
