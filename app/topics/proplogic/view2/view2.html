Tree-based formula evaluation

Given the following variable assignments:

  • `a rarr 0`
  • `b rarr 1`
  • `c rarr 1`
  • `d rarr 1`

The following tree represents the boolean formula `(a ^^ b) vv (c o+ D)`. Please assign the values of the atoms and operators to the respective boxes.

`vv`
`^^`
`o+`
`a`
`b`
`c`
`d`

Answer is correct.

Answer is wrong. Please take a look at the following truth table for the `vv` operator:

`vv` 0 1
0 0 1
1 1 1


Storyboard

Goal: This exercise demonstrates the evaluation of propositional logic in tree form. The advantage over the exercise 1 is, that the operator precedence and parentheses are already clear through the tree form, so we can check directly if the student is able to evaluate the logical operators.

Usage: Please type `1` into the box of the `vv` operator and press check answer. You will the that the truth table for the `vv`-operator is shown since the expected answer is 0.

Other: For each operator that was evaluated incorrectly we keep a record of how many times the student made a mistake with that operator. Then we can adjust further questions to explicitly cover this operator.

Note: In the final version we plan to draw the tree on the left in a more graph-like fashion by using an appropriate drawing library.
