Storyboard

This exercise demonstrates the evaluation of propositional logic in tree form. The advantage over the exercise 1 is, that the operator precedence and parentheses are already clear through the tree form, so we can check directly if the student is able to evaluate the logical operators.

Assuming the student types 1 into the box of "v" (or operator), then we show the truth table for "OR" and let the student evaluate the question again.

For each operator that was evaluated incorrectly we keep a record of how many times the student made a mistake with that operator. Then we can adjust further questions to explicitly cover this operator.
