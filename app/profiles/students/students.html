Storyboard

This is the profile view for the student. It includes information about the student. There will not be an overall progress for the student since progress can only be made in the individual lectures and their respective units.

For now we do not include gamification elements but they could possibly be added later.

For the student we are showing some data of his profile such as:

  • Information about the student, e.g. name, username, avatar icon
  • Clicking on "Change avatar" will open a dialog where the user can upload a new avatar icon or select one of several predefined icons.
  • For the "edit profile" button there are still some open questions:
    • Should the user be able to change his full name?
    • Should the user be able to change his Matrikelnumber?
    • Are there any other profile related settings that we might want to let the student edit?
    • Depending on how we decide this, the "edit profile" button could either be removed again, or it could open a dialog where the user can make the respective adjustments to his profile.
